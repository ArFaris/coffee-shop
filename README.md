# Coffee Shop Project

Это проект веб-сайта магазина с кофе. Основной упор в разработке сделан на модульность и организованность стилей, что позволяет легко масштабировать и поддерживать проект.

## Ссылки
* **Демо (GitHub Pages):** [Public page](https://arfaris.github.io/coffee-shop/)

---

## Технологии
* **HTML5** (Семантическая верстка)
* **Modular CSS** (Чистый CSS с разделением на компоненты и секции)

---

## Структура проекта (CSS)

В проекте реализована модульная архитектура стилей. Основные файлы расположены в директории `src/styles`:

### 1. Base (`/base`)
Базовые настройки, которые применяются ко всему проекту:
* `variables.css` — Глобальные CSS-переменные (цвета, шрифты, размеры).
* `reset.css` — Сброс стандартных стилей браузера.
* `fonts.css` — Подключение и настройка шрифтов.
* `base.css` — Общие стили для тегов (body, links).

### 2. Components (`/components`)
Изолированные стили для переиспользуемых мелких элементов интерфейса:
* `Buttons/` — Стили кнопок.
* `CoffeeCard/` — Карточки товаров.
* `TitleArea/` — Оформление заголовков и текстовых блоков.
* `CoffeeLine/`, `Footer/`, `HeaderNav/` и др.

### 3. Sections (`/sections`)
Стили для крупных блоков страницы:
* `BaseInfo/` — Информационный блок.
* `CoffeeSection/` — Секция с ассортиментом кофе.
* `GiftsetSection/` — Подарочные наборы.

### 4. Utils & Entry Point
* **`utils/animations.css`** — Описание ключевых кадров (`keyframes`) и вспомогательных классов для анимации.
* **`main.css`** — Главный файл, который импортирует в себя все модули (точка входа).
